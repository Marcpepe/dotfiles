# dotfiles
> command line productivity tools

## Tools
- git
- tmux
- vim
- zsh

## Steps
- Install git / tmux / vim / zsh
- Clone the project : `git clone https://github.com/vbrajon/dotfiles .dotfiles`
- Backup old config
- Make symbolic links : `cd .dotfiles && ./init.sh`
- Set zsh as default shell : `chsh`
